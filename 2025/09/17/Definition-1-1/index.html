Definition 1.1

发表于2025-09-17|更新于2025-09-17|课程笔记
|总字数:358|阅读时长:1分钟|浏览量:|评论数:

该定义定义了什么是极小值点，定义如下：

A point x* ∈ Ω is called a local minimum point of f over Ω if there exists δ > 0 such that f(x*) ≤ f(x) ∀ x ∈ Ω with ||x − x*|| < δ.

通俗理解

设想我们是一座山区中的一个点，现在我们要判断我们所在的位置 x* 是不是一个“局部山谷”，即局部最低点。

我们来一条一条拆解上面的定义：

  • x* ∈ Ω ：我们（一个点）必须站在我们所关心的这一座山区中（Ω）。
  • if there exists δ > 0 such that…：存在一个距离（比如10米），让我们能画出一个圈子。这个圈子不用很大，哪怕只是一个非常小的范围（比如周围 1 米）也足够。关键是​​存在这么一个圈子​​。
  • ||x − x*|| < δ ：这是用来确定“圈子”里的点。||x − x*|| 表示另一个点 x 和我们所在的位置 x* 之间的距离（就像我们之前讨论的，这是向量的长度/距离）。 < δ 意味着所有和我们距离小于 δ 的点都在这个圈子里。
  • f(x*) ≤ f(x) ：这是核心条件，它要求我们的海拔 f(x*) 必须小于或等于圈子里所有其它点 x 的海拔 f(x) 。
  • ∀ x ∈ Ω with ||x − x*|| < δ ：上面的四条必须对圈子里的每一个点都成立，不能有一个例外。
