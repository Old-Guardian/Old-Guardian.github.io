Theorem 1.8 & 1.9

发表于2025-10-27|更新于2025-10-27|课程笔记
|总字数:677|阅读时长:2分钟|浏览量:|评论数:

本篇主要介绍关于凸函数的性质。

Theorem 1.8

For a convex function, we have the following statements:

  1. A function f defined on a convex set Ω is convex if and only if for any integer k ≥ 2, any x1, x2, ..., xk ∈ Ω, and any non-negative α1, ..., αk with $\sum_{i=1}^k \alpha_i = 1$, there holds

$$ f(\sum_{i=1}^k \alpha_i \mathbf{x}_i ) \le \sum_{i=1}^k \alpha_i f(\mathbf{x}_i) $$

  1. Let f be a convex function defined on a convex set Ω . Then f is continuous at every interior point of Ω.

(i)

我们先理解表达式的各个部分：

  • $\sum_{i=1}^k \alpha_i \mathbf{x}_i$ ：这是多个点的凸组合。
  • $\sum_{i=1}^k \alpha_i f(\mathbf{x}_i)$ ：这是函数值的加权平均。

所以，这个不等式的含义就是：函数在凸组合点的值 ≤ 函数值的凸组合。

这个不等式叫做 Jensen 不等式，它实际上是 Definition 1.6 中凸函数定义的多元推广。

几何上来说，

  • 点 $\sum_{i=1}^k \alpha_i \mathbf{x}_i$ 位于这些点构成的凸包内；
  • $\sum_{i=1}^k \alpha_i f(\mathbf{x}_i)$ 是连接各点 (xi, f(xi)) 的超平面在该点的值。
  • 不等式说明：函数的图像在整个凸包内部都位于支撑超平面的下方。

Theorem 1.9

Let f and g be two convex functions on the convex set Ω. Then the following statements hold:

  1. f + g is convex on Ω.
  2. cf is convex for any constant c > 0.
  3. Γ = {x : x ∈ Ω and f(x) ≤ c} is a convex set for any c.

(iii)

第三条描述的是凸函数的水平集（level set） 的重要性质。

这个定理说的是，对于凸函数 f 和任意常数 c ，集合 Γ = {x : x ∈ Ω and f(x) ≤ c} 是一个凸集。

这个凸集被称为函数 f 的 c 水平集，它包含了所有函数值不超过 c 的点。

我们可以想象：

  • 想象一个凸函数的图像，像一个碗形曲面。
  • 用一个水平平面 z = c 去切割这个曲面。
  • 切割后，曲面上所有低于或等于这个平面的点构成的集合就是 Γ 。
  • 定理说：这个切割出来的区域一定是凸的。

这个定理的重要意义之一在于构造可行域。

在约束优化问题中，我们常常处理这样的问题：

$$ \begin{aligned} &\text{Minimize}\enspace g(x) \\ &\text{subject to}\enspace f_i(x)\le 0,\enspace i=1,\ldots,m \end{aligned} $$

如果所有约束函数 fi 都是凸函数，那么：

  • 每个约束 fi(x) ≤ 0 实际上都定义了一个凸集（取 c = 0 的水平集）。
  • 可行域是这些凸集的交集。
  • 凸集的交集仍是凸集，所以整个可行域是凸的。

这就保证了我们处理的是凸优化问题，具有很好的性质（局部最优=全局最优）。
