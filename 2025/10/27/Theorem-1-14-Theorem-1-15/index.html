Theorem 1.14 & Theorem 1.15

Theorem 1.14

该定理给出了凸函数的一阶必要条件。

Let f ∈ C1 be convex on the convex set Ω. If there exists x* ∈ Ω such that

∇f(x*)T(x − x*) ≥ 0  ∀x ∈ Ω

then x* is a global minimum of f over Ω.

几何含义

我们来拆解这个不等式：

  • ∇f(x*)：函数在点 x* 处的梯度（函数上升最快的方向）；
  • x − x*：从点 x* 指向 Ω 中任意点 x 的向量；
  • ∇f(x*)T(x − x*)：梯度在方向 x − x* 上的投影。

这个不等式的含义就是：从 x* 出发，沿着指向 Ω 中任何其他点的方向，函数都不会下降。

这实际上与我们之前介绍的 Theorem 1.5 一阶必要条件是一样的，我们同样可以考虑两种特殊情况：

  • x* 在 Ω 内部；
  • x* 在 Ω 边界上；

分析过程也与一阶必要条件相同：

  • 如果 x* 在凸集内部，那么我们可以朝任何方向移动：
    • 对于任意方向 d ，取 x = x* + ϵd （小步长）；
    • 条件要求 ∇f(x*)T(x − x*) ≥ 0 对所有 d 都成立；
    • 这只能发生在 ∇f(x*) = 0 的时候，原因见 Theorem 1.5 。
  • 如果 x* 在凸集边界上，要让 ∇f(x*)T(x − x*) ≥ 0  ∀x ∈ Ω 成立的话，所有可行方向都要与梯度成锐角或直角。

与凸函数一阶最优性条件的联系

这个定理非常好用 Theorem 1.10 证明：

我们从 Theorem 1.10 凸函数的一阶最优性条件中有

f(y) ≥ f(x) + ∇f(x)T(y − x)

我们把 ∇f(x*)T(x − x*) ≥ 0 代入上式，就能得到

f(y) ≥ f(x) + ∇f(x)T(y − x) ≥ f(x*)

所以，对于任意的 x ∈ Ω, 有 f(y) ≥ f(x*) ，即 x* 是全局最小值点。

与一阶必要条件的联系

还记得我们在 Theorem 1.5 中给出的一阶必要条件：

Let f ∈ C1 be a function on Ω ⊆ ℝn. If x* is a local minimum point of f over Ω, then

∇f(x*)Td ≥ 0

for any feasible direction d ∈ ℝn at x*.

可以看到，二者的形式是高度一致的。实际上，我们有如下结论：

For convex functions, the first-order conditions are both necessary and sufficient for a point to be a global minimum.

即对于凸函数，一阶必要条件是充要的：

  • 如果 x* 是全局最小值点，那么它一定满足一阶必要条件；
  • 如果 x* 满足一阶必要条件，那么它一定是全局最小值点。

相比之下，对于非凸函数，就如我们之前介绍的那样：

  • 如果 x* 是局部最小值点，那么它一定满足一阶必要条件；
  • 如果 x* 满足一阶必要条件，它不一定是局部最小值点，还可能是鞍点或局部最大值。

几何上也很直观：

  • 凸函数就像是一个碗：
    • 如果碗底是平的（梯度为零），那么这就是最低点；
    • 由于碗的形状是”向上弯曲的”，不可能在平点之外还有更低点；
    • 平点自动就是整个碗的最低点。
  • 非凸函数就像是一个山地：
    • 一个平坦点（梯度为零）可能只是小山谷的底部；
    • 周围可能有更深的山谷；
    • 所以零梯度不能保证全局最优。

Theorem 1.15

该定理指出了凸函数的最大值会在凸集上的极值处达到。

Let f be a convex function defined on a bounded, closed convex set Ω. If f has a maximum value over Ω , then it is achieved at an extreme point of Ω.

这个定理主要说的是：当你在一个“形状规则”的凸区域（有界闭凸集）上，对一个凸函数寻找最大值时，你不需要搜索整个区域，只需要检查区域的“边界”上的那些“最突出的点”（极点）就可以了。
