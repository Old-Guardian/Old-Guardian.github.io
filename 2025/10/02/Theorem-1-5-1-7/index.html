Theorem 1.5-1.7

发表于2025-10-02|更新于2025-10-02|课程笔记
|总字数:1.8k|阅读时长:5分钟|浏览量:|评论数:

本篇分别记述了优化理论中的三大核心定理：一阶必要条件（FNC），二阶必要条件（SNC）和二阶充分条件（SSC）

Theorem 1.5

一阶必要条件是优化理论中的核心定理之一，它描述了局部最小点必须满足的条件。

Let f ∈ C1 be a function on Ω ⊆ ℝn. If x* is a local minimum point of f over Ω, then ∇f(x*)Td ≥ 0 for any feasible direction d ∈ ℝn at x*.

代数解释

这个定理的核心是 ∇f(x*)Td ，也就是梯度向量 ∇f(x*)T 与可行方向 d 的点积，实际上就是函数 f 在点 x* 处沿方向 d 的方向导数，表示函数在该方向上的瞬时变化率。

定理要求，如果 x* 是 f 在 Ω 上的局部最小点，那么对于任何可行方向 d ，方向导数必须满足 ∇f(x*)Td ≥ 0 。

这实际上意味着：在局部最小点处，函数沿任何可行方向都不会立即下降（即方向导数非负）。如果存在一个可行方向 d 使得方向导数为负，那么沿着 d 移动一个很小的步长，函数值会减少，这与 x* 是局部最小点矛盾。

以点 x* 的位置区分，有两种情况：

  1. 如果点 x* 是 Ω 的内点，也就是存在一个邻域完全包含在 Ω 内，那么所有方向都是可行方向，例如下图：

图中的球体就是 x* 的一个邻域，可以看到对于 x* 而言，所有方向都是可行方向。因此，此时对于任何 d ，都有 ∇f(x*)Td ≥ 0 。

这实际上等价于 ∇f(x*)Td = 0 ，因为只有当 ∇f(x*) = 0 时，∇f(x*)Td ≥ 0 才有可能成立。因为如果梯度非零，取 d = −∇f(x*) 就会使点积为负。

所以，在内点处，一阶必要条件就简化为梯度为零的条件。

这对应 Corollary 1.5.1: Unconstrained Case，指的就是 x 不在约束边界上的情况。

  1. 如果点 x* 在 Ω 的边界上，则只有某些方向是可行的，比如指向 Ω 内部的方向：

此时，梯度不一定为零，但必须与所有可行方向的点积非负，也就是满足原定理 ∇f(x*)Td ≥ 0 。

几何解释

我们知道，梯度向量 ∇f(x*) 指向函数增长最快的方向，也就是函数曲面在点 x* 处的法向量。而可行方向 d 可以视为从 x* 出发，与可行域 Ω 相切的方向。

这样来看，上述定理的几何含义就是：

  • 在局部最小点 x* 处，梯度向量必须与所有可行方向成锐角或直角（即点积非负）。这意味着梯度向量必须指向可行域 Ω 的内部，或者至少与可行方向的垂直。
  • 如果梯度指向可行域的外部，此时必然存在一个可行方向与梯度成钝角（使得点积为负），从而函数值会下降，这就与这与局部最小点矛盾。因此，梯度方向必须“支持”可行域——在边界点上，梯度往往与边界垂直并指向外部，但由于约束，可行方向只能指向内部，所以点积非负。

总之，一阶必要条件是一个基本工具，它将局部最优性与梯度方向和可行方向联系起来。无论是代数还是几何视角，都强调了在局部最小点处，函数不能有下降的可行方向。

Theorem 1.6

在一阶必要条件的基础上，二阶必要条件提供了更深入和精细的判据。

Let f ∈ C2 be a function on Ω ⊆ ℝn and let x* be a local minimum point. Then for any feasible direction d ∈ ℝn at x* , we have

  1. ∇f(x*)Td ≥ 0
  2. dT∇2f(x*)d ≥ 0 whenever ∇f(x*)Td = 0

还记得我们之前提到过，∇2f(x*) 是函数 f 在点 x* 处的海森矩阵(Hessian Matrix)，由于 d 是向量，所以二阶必要条件可以翻译成

如果 f 是二次连续可微的，且 x* 是一个局部最小值点，那么对于任意 x* 处的可行方向 d ，我们有

  1. 一阶必要条件成立。
  2. 若 ∇f(x*)Td = 0 ，则==海森矩阵一定是半正定(Positive Definite Matrix)的。==

第二个条件非常重要，我们会一次又一次地用到它，参见 Corollary 1.6.1.

意义

二阶必要条件最重要的作用就是排除鞍点：

如果一个临界点的海森矩阵不是半正定的，那么它不可能是局部最小值。

Theorem 1.7

二阶充分条件为我们确定严格局部极小值点提供了依据。

Let f ∈ C2 be a function on Ω ⊆ ℝn and let x* be an interior point of Ω . If

  1. ∇f(x*) = 0 and
  2. ∇2f(x*) is positive definite,

then x* is a strict local minimum point of f .

与二阶必要条件的区别

条件 二阶必要条件 二阶充分条件
数学表述 ∇f(x*) = 0 且 ∇2f(x*) 半正定 ∇f(x*) = 0 且 ∇2f(x*) 正定
逻辑关系 如果 x* 是局部极小点，则必须满足此条件 如果满足此条件，则 x* 一定是严格局部极小点
确定性 是必要但不充分的：满足条件的点可能是极小点，也可能是其他点 是充分但不必要的：满足条件的点一定是极小点，但有些不满足条件的点也可能是极小点

在实践中，二阶必要条件的作用是排除非候选点：

  • 这是一个”否决性”条件。
  • 如果我们计算一个临界点（∇f(x*) = 0）的海森矩阵，发现它是不定的（有负特征值），那么我们可以立即断定这个点不是局部极小点（比如是鞍点）。
  • 但它不能确认极小点：即使海森矩阵是半正定的，这个点也可能不是极小点（如例 1）。

而二阶充分条件的作用是确认极小值点：

  • 这是一个”确认性”条件。
  • 如果我们计算一个临界点的海森矩阵，发现它是正定的，那么我们可以立即断定这个点一定是严格的局部极小点。
  • 但它不是唯一途径：有些局部极小点可能不满足这个条件（如例 2 ）。

例1

考虑函数 f(x) = x3 在 x = 0 处：

  • ∇f(0) = 0 ，满足一阶必要条件。
  • ∇2f(0) = 0 ，零矩阵是半正定的，所以也满足二阶必要条件。
  • 但我们看图像就知道，x = 0 并不是极值点，而是拐点。

这说明，满足二阶必要条件的点不一定是最小点。

例2

考虑函数 f(x) = x4 在 x = 0 处：

  • ∇f(0) = 0 ，满足一阶必要条件。
  • ∇2f(0) = 0 ，零矩阵是半正定的，所以不满足二阶充分条件。
  • 但我们看图像就知道，x = 0 确实是严格的局部极小值点。

这说明，不满足二阶充分条件的点也可能是极小点。
